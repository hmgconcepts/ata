## 🌾 Platform Overview
**The Ata v2** is an expanded, multi-page enterprise agri-tech platform built to address Nigeria's **₦5 Trillion annual post-harvest loss**. Engineered following deep market benchmarking against global industry leaders (**ProducePay**, **Silo.co**, **ColdHubs**, **Ninjacart**, **Zowasel**), Ata v2 combines direct farm-to-buyer transactions, IoT vehicle telemetry, Quick-Pay invoice factoring, pay-as-you-store solar cold room booking, national commodity price indices, and escrow capital security.

## 🚀 Quick Start (Local Run)

```bash
# Navigate to project directory
cd "ata v2"

# Start static server using npm
npm start

# Or using Python
python3 -m http.server 3000
```
Open `http://localhost:3000` in your web browser.
